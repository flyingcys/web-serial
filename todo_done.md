## 📋 使用指南

### 快速部署
1. 下载所有文件到本地目录
2. 在Chrome/Edge浏览器中打开`index.html`
3. 确保使用HTTPS协议（网络部署时）
4. 开始串口调试

### 开发环境要求
- Node.js（可选，用于本地服务器）
- Chrome 89+ 或 Edge 89+
- HTTPS环境（生产环境）
