# Chrome 串口调试助手功能规划

## 项目概述

开发一个基于 Chrome 浏览器的串口调试助手，使用 Web Serial API 实现串口通信功能。

## 核心功能模块

### 1. 串口连接管理

#### 1.1 串口配置
- [x] 波特率选择（常用：9600, 19200, 38400, 57600, 115200 等）
- [x] 数据位配置（5, 6, 7, 8 位）
- [x] 停止位配置（1, 1.5, 2 位）
- [x] 校验位配置（无, 奇校验, 偶校验）
- [x] 流控制配置（无, RTS/CTS, XON/XOFF）
- [x] 目标设备选择（下拉菜单）

#### 1.2 连接控制
- [x] 连接串口按钮
